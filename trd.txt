PART ONE: SYSTEM ARCHITECTURE
Section 1: Architecture Overview
1.1 Architecture Philosophy
VectorShift follows a data-first, pre-computation architecture. The most important architectural decision is this: nothing is calculated at request time that can be calculated in advance.

This means:

All suitability scores are pre-calculated during data ingestion
All vulnerability indexes are stored, not derived on the fly
Country profile pages are statically generated at build time
The map loads pre-aggregated GeoJSON with scores embedded
This approach delivers:

Sub-100ms page loads for most content
Zero database load during peak traffic
Full functionality even if the database is temporarily slow
No expensive real-time computation on free-tier infrastructure

1.3 Architecture Decision Records
These are explicit decisions made and their reasoning:

Decision	Choice Made	Reason	Alternative Rejected
Framework	Next.js 14 App Router	SSG + ISR + API routes in one framework. Best free hosting on Vercel	Separate Express backend — adds complexity
Database	Supabase PostgreSQL	Free tier, PostGIS included, real-time subscriptions, auth built-in	PlanetScale — no PostGIS. MongoDB — wrong for relational data
Map library	Mapbox GL JS	Best performance for choropleth at world scale, free tier sufficient, excellent customization	Leaflet — slower for large GeoJSON. DeckGL — too complex
Charts	Recharts primary, D3 for complex	Recharts for standard charts (fast), D3 for custom visualizations (powerful)	Chart.js — less control. Victory — less active
Styling	Tailwind CSS	Utility-first, no runtime CSS, excellent with shadcn	CSS Modules — more verbose. Styled Components — runtime overhead
Components	shadcn/ui	Unstyled, copy-paste, full control, no lock-in	Chakra UI — too opinionated. MUI — heavy
AI	Groq API (free tier)	Fastest inference (300+ tokens/sec), generous free tier, no credit card required	OpenAI — paid. Gemini — good alternative
ETL language	Python	Best data science ecosystem, pandas for data processing, requests for APIs	Node.js — inferior data processing libraries
Deployment	Vercel	Free tier, automatic deployments, ISR support, edge functions	Netlify — good but Vercel + Next.js is first-party
State management	Zustand	Lightweight, no boilerplate, perfect for map/filter state	Redux — overkill. Context only — performance issues with map

4.3 Complete API Route Specifications
Route 1: GET /api/map
Purpose: Returns GeoJSON FeatureCollection for all countries with suitability scores embedded as properties. This is the primary data feed for the Mapbox map.

Parameters:

Parameter	Type	Required	Default	Validation
disease	string	Yes	—	One of 6 valid disease slugs
year	integer	Yes	—	1981–2050
scenario	string	No	null	SSP1-2.6, SSP2-4.5, SSP5-8.5 (required if year > 2023)

Implementation Logic:

// src/app/api/map/route.ts

import { NextRequest, NextResponse } from 'next/server'
import { z } from 'zod'
import { createServerSupabaseClient } from '@/lib/supabase/server'
import { VALID_DISEASES, VALID_SCENARIOS } from '@/lib/constants/diseases'

const QuerySchema = z.object({
  disease: z.enum(VALID_DISEASES),
  year: z.coerce.number().int().min(1981).max(2050),
  scenario: z.enum(VALID_SCENARIOS).optional().nullable(),
})

export async function GET(request: NextRequest) {
  const { searchParams } = new URL(request.url)
  
  // Validate input
  const parsed = QuerySchema.safeParse({
    disease: searchParams.get('disease'),
    year: searchParams.get('year'),
    scenario: searchParams.get('scenario'),
  })
  
  if (!parsed.success) {
    return NextResponse.json(
      { success: false, error: parsed.error.errors[0] },
      { status: 400 }
    )
  }
  
  const { disease, year, scenario } = parsed.data
  
  // Enforce scenario requirement for projected years
  if (year > 2023 && !scenario) {
    return NextResponse.json(
      { success: false, error: { code: 'SCENARIO_REQUIRED', 
        message: 'scenario parameter required for years after 2023' }},
      { status: 400 }
    )
  }
  
  const supabase = createServerSupabaseClient()
  
  // Single optimized query joining countries + scores + population
  const { data, error } = await supabase.rpc('get_map_data', {
    p_disease: disease,
    p_year: year,
    p_scenario: scenario ?? null
  })
  
  if (error) {
    return NextResponse.json(
      { success: false, error: { code: 'DB_ERROR', message: error.message }},
      { status: 500 }
    )
  }
  
  // Transform to GeoJSON FeatureCollection
  const geojson = buildFeatureCollection(data)
  
  return NextResponse.json(
    { success: true, data: geojson, meta: { disease, year, scenario } },
    {

CREATE OR REPLACE FUNCTION get_map_data(
  p_disease TEXT,
  p_year SMALLINT,
  p_scenario TEXT DEFAULT NULL
)
RETURNS TABLE (
  iso3 CHAR(3),
  iso2 CHAR(2),
  name VARCHAR,
  slug VARCHAR,
  geojson JSONB,
  score DECIMAL,
  risk_level VARCHAR,
  is_newly_at_risk BOOLEAN,
  score_change_since_2000 DECIMAL,
  population BIGINT
) AS $$
BEGIN
  RETURN QUERY
  SELECT 
    c.iso3,
    c.iso2,
    c.name,
    c.slug,
    c.geojson,
    ss.suitability_score,
    ss.risk_level,
    ss.is_newly_at_risk,
    ss.score_change_since_2000,
    COALESCE(pd.total_population, 0)
  FROM countries c
  LEFT JOIN diseases d ON d.slug = p_disease
  LEFT JOIN suitability_scores ss ON 
    ss.country_id = c.id AND 
    ss.disease_id = d.id AND 
    ss.year = p_year AND
    (ss.scenario IS NULL AND p_scenario IS NULL OR ss.scenario = p_scenario)
  LEFT JOIN population_data pd ON 
    pd.country_id = c.id AND 
    pd.year = LEAST(p_year, 2023)  -- Use latest available for future years
  ORDER BY c.name;
END;

import Groq from 'groq-sdk'
import { buildDatabaseContext } from '@/lib/ai/context-builder'
import { SYSTEM_PROMPT } from '@/lib/ai/system-prompt'
import { ratelimit } from '@/lib/rate-limiter'

const groq = new Groq({ apiKey: process.env.GROQ_API_KEY })

export async function POST(request: NextRequest) {
  // Rate limiting: 10 requests per minute per IP
  const ip = request.ip ?? 'unknown'
  const { success: allowed } = await ratelimit.limit(ip)
  
  if (!allowed) {
    return NextResponse.json(
      { success: false, error: { code: 'RATE_LIMITED', 
        message: 'Too many requests. Please wait a moment.' }},
      { status: 429 }
    )
  }
  
  const body = await request.json()
  const { query, session_id, context } = body
  
  // Validate query
  if (!query || query.length > 500) {
    return NextResponse.json(
      { success: false, error: { code: 'INVALID_QUERY' }},
      { status: 400 }
    )
  }
  
  // Build database context (query relevant data based on question)
  const dbContext = await buildDatabaseContext(query, context)
  
  // Call Groq
  const completion = await groq.chat.completions.create({
    model: 'llama-3.1-70b-versatile',
    messages: [
      { role: 'system', content: SYSTEM_PROMPT },
      { role: 'user', content: `
        DATABASE CONTEXT:
        ${JSON.stringify(dbContext, null, 2)}
        
        USER QUESTION:
        ${query}
      `}
    ],
    max_tokens: 600,
    temperature: 0.3,  // Lower temperature for factual responses
  })
  
  const answer = completion.choices[0]?.message?.content
  
  return NextResponse.json({
    success: true,
    data: {
      answer,
      sources: dbContext.sources,
      related_links: dbContext.related_links
    }
  })
}

// Analyzes the user's question and queries the most relevant data

export async function buildDatabaseContext(
  query: string, 
  userContext?: UserContext
) {
  const supabase = createServerSupabaseClient()
  
  // Simple keyword detection to determine what data to fetch
  const mentionedCountries = extractCountries(query)    // e.g., ['egypt', 'italy']
  const mentionedDiseases = extractDiseases(query)      // e.g., ['dengue']
  const mentionedYears = extractYears(query)            // e.g., [2024, 2050]
  
  const contextData: any = { sources: [], related_links: [] }
  
  // Fetch relevant country scores
  if (mentionedCountries.length > 0) {
    const { data } = await supabase
      .from('suitability_scores')
      .select(`
        suitability_score, risk_level, year, score_change_since_2000,
        diseases(slug, name),
        countries(name, iso3, slug)
      `)
      .in('countries.slug', mentionedCountries)
      .in('diseases.slug', mentionedDiseases.length > 0 
        ? mentionedDiseases 
        : VALID_DISEASES)
      .eq('year', 2024)
      .is('scenario', null)
      .limit(20)
    
    contextData.country_scores = data
    contextData.related_links = mentionedCountries.map(c => ({
      text: `View ${c} profile`,
      href: `/country/${c}`
    }))
  }
  
  // Add global context if general question
  if (mentionedCountries.length === 0) {
    const { data } = await supabase
      .from('global_stats_cache')
      .select('*')
      .eq('year', 2024)
      .is('scenario', null)
      .limit(6)
    
    contextData.global_stats = data
  }
  
  contextData.sources = [
    'NASA POWER climate data',
    'WHO Global Health Observatory',
    'Published vector biology literature'
  ]
  
  return contextData
}

    """

def collect_all_countries():

    for i, country in enumerate(COUNTRIES):
        iso3 = country['iso3']
        output_file = output_dir / f"{iso3}.json"
        
        # Skip if already collected
        if output_file.exists():
            print(f"[{i+1}/195] {iso3} — cached, skipping")
            continue
        
        print(f"[{i+1}/195] Collecting {iso3}...")
        
        for attempt in range(3):  # 3 retries
            try:
                data = collect_country_climate(iso3)
                output_file.write_text(json.dumps(data, indent=2))
                print(f"  ✓ {iso3} saved")
                time.sleep(1)  # Respectful rate limiting
                break
            except Exception as e:
                if attempt == 2:
                    print(f"  ✗ {iso3} failed after 3 attempts: {e}")
                    failed.append(iso3)
                else:
                    print(f"  Retry {attempt+1}...")
                    time.sleep(5)
    
    if failed:
        print(f"\nFailed countries: {failed}")
        Path("raw_climate/failed.txt").write_text("\n".join(failed))
    
    print(f"\nCollection complete. {len(COUNTRIES) - len(failed)}/195 successful.")

# etl/processors/score_calculator.py

import pandas as pd
import numpy as np
from config.diseases import DISEASE_PARAMETERS

def calculate_temp_score(
    temp_mean: float, 
    temp_min_coldest: float,
    params: dict
) -> tuple[float, float]:
    """
    Returns (temp_score, winter_score) both in range 0.0-1.0
    """
    t = params['temperature']
    
    # Temperature suitability score
    if temp_mean < t['min'] or temp_mean > t['max']:
        temp_score = 0.0
    elif t['optimal_min'] <= temp_mean <= t['optimal_max']:
        temp_score = 1.0
    elif temp_mean < t['optimal_min']:
        temp_score = (temp_mean - t['min']) / (t['optimal_min'] - t['min'])
    else:
        temp_score = (t['max'] - temp_mean) / (t['max'] - t['optimal_max'])
    
    # Winter survival score (can the vector overwinter?)
    winter_min = t['winter_min']
    if temp_min_coldest < winter_min:
        winter_score = 0.0
    elif temp_min_coldest > winter_min + 5:
        winter_score = 1.0
    else:
        winter_score = (temp_min_coldest - winter_min) / 5.0
    
    return round(max(0.0, min(1.0, temp_score)), 3), \
           round(max(0.0, min(1.0, winter_score)), 3)


def calculate_humidity_score(humidity_mean: float, params: dict) -> float:
    h = params['humidity']
    if humidity_mean < h['min']:
        return 0.0
    elif humidity_mean > h['min'] + 20:
        return 1.0
    else:
        return round((humidity_mean - h['min']) / 20.0, 3)


def calculate_rain_score(suitable_months: int, params: dict) -> float:
    required = params['precipitation']['required_months']
    if suitable_months < required:
        return 0.0
    return round(min(1.0, suitable_months / (required * 1.5)), 3)


def calculate_altitude_score(altitude_median: float, params: dict) -> float:
    max_alt = params['altitude']['max']
    if altitude_median > max_alt:
        return 0.0
    elif altitude_median > max_alt * 0.8:
        return 0.5
    else:
        return 1.0


def calculate_suitable_months(
    monthly_precip: list[float], 
    min_monthly: float
) -> int:
    """Count months where precipitation exceeds threshold."""
    return sum(1 for p in monthly_precip if p >= min_monthly)


def calculate_suitability(
    climate_row: dict,
    disease_slug: str
) -> dict:
    """
    Calculate all score components and final suitability score
    for one country, one year, one disease.
    """
    params = DISEASE_PARAMETERS[disease_slug]
    
    monthly_precip = [
        climate_row.get(f'precip_monthly_{m}', 0) or 0
        for m in ['jan','feb','mar','apr','may','jun',
                  'jul','aug','sep','oct','nov','dec']
    ]
    
    suitable_months = calculate_suitable_months(
        monthly_precip, 
        params['precipitation']['min_monthly']
    )
    
    temp_score, winter_score = calculate_temp_score(
        climate_row['temp_mean_annual'],
        climate_row['temp_min_coldest'],
        params
    )
    
    humidity_score = calculate_humidity_score(
        climate_row['humidity_mean'],
        params
    )
    
    rain_score = calculate_rain_score(suitable_months, params)
    
    altitude_score = calculate_altitude_score(
        climate_row['altitude_median'],
        params
    )
    
    # Weighted composite
    final_score = (
        temp_score * 0.35 +
        winter_score * 0.25 +
        humidity_score * 0.20 +
        rain_score * 0.15 +
        altitude_score * 0.05
    )
    
    final_score = round(max(0.0, min(1.0, final_score)), 3)
    
    # Risk level classification
    if final_score >= 0.8:
        risk_level = 'critical'
    elif final_score >= 0.6:
        risk_level = 'high'
    elif final_score >= 0.4:
        risk_level = 'moderate'
    elif final_score >= 0.2:
        risk_level = 'low'
    else:
        risk_level = 'none'
    
    return {
        'temp_score': temp_score,
        'winter_score': winter_score,
        'humidity_score': humidity_score,
        'rain_score': rain_score,
        'altitude_score': altitude_score,
        'suitability_score': final_score,
        'risk_level': risk_level,
        'suitable_months': suitable_months
    }


def calculate_all_scores(climate_df: pd.DataFrame, countries_df: pd.DataFrame) -> pd.DataFrame:
    """
    Calculate suitability scores for all countries × years × diseases.
    Returns a DataFrame ready for database insertion.
    """
    results = []
    diseases = list(DISEASE_PARAMETERS.keys())
    
    for _, country in countries_df.iterrows():
        country_climate = climate_df[
            climate_df['country_id'] == country['id']
        ].copy()
        
        # Merge altitude from countries table
        country_climate['altitude_median'] = country['altitude_median']
        
        for year_data in country_climate.itertuples():
            for disease in diseases:
                try:
                    scores = calculate_suitability(
                        year_data._asdict(), 
                        disease
                    )
                    results.append({
                        'country_id': country['id'],
                        'disease_slug': disease,
                        'year': year_data.year,
                        'scenario': None,
                        **scores
                    })
                except Exception as e:
                    print(f"Error: {country['iso3']} {year_data.year} {disease}: {e}")
    
    df = pd.DataFrame(results)
    
    # Calculate year-over-year changes
    df = df.sort_values(['country_id', 'disease_slug', 'year'])
    df['score_change_1yr'] = df.groupby(
        ['country_id', 'disease_slug']
    )['suitability_score'].diff(1)
    df['score_change_5yr'] = df.groupby(
        ['country_id', 'disease_slug']
    )['suitability_score'].diff(5)
    
    # Score change since 2000
    scores_2000 = df[df['year'] == 2000][
        ['country_id', 'disease_slug', 'suitability_score']
    ].rename(columns={'suitability_score': 'score_2000'})
    
    df = df.merge(scores_2000, on=['country_id', 'disease_slug'], how='left')
    df['score_change_since_2000'] = df['suitability_score'] - df['score_2000']
    df.drop(columns=['score_2000'], inplace=True)
    
    # Flag newly at risk (crossed 0.4 threshold in last 5 years)
    def is_newly_at_risk(group):
        group = group.sort_values('year')
        recent = group[group['year'] >= group['year'].max() - 5]
        past = group[group['year'] < group['year'].max() - 5]
        if len(past) == 0:
            return pd.Series([False] * len(group), index=group.index)
        
        was_below = (past['suitability_score'] < 0.4).all()
        is_above = (recent['suitability_score'] >= 0.4).any()
        newly = was_below and is_above
        return pd.Series([newly] * len(group), index=group.index)
    
    df['is_newly_at_risk'] = df.groupby(
        ['country_id', 'disease_slug'], 
        group_keys=False
    ).apply(is_newly_at_risk)
    
    return df

// src/components/map/GlobalMap.tsx
// This is the most complex component. Full specification below.

'use client'

import { useEffect, useRef, useCallback } from 'react'
import mapboxgl from 'mapbox-gl'
import 'mapbox-gl/dist/mapbox-gl.css'
import { useMapStore } from '@/hooks/useMapStore'
import { useMapData } from '@/hooks/useMapData'
import { MAPBOX_STYLE_URL } from '@/lib/mapbox/config'
import { buildRiskColorExpression } from '@/lib/mapbox/colors'
import { CountryPanel } from './CountryPanel'
import { DiseaseSelector } from './DiseaseSelector'
import { TimeSlider } from './TimeSlider'
import { MapLegend } from './MapLegend'
import { MapControls } from './MapControls'
import { ScenarioToggle } from './ScenarioToggle'

mapboxgl.accessToken = process.env.NEXT_PUBLIC_MAPBOX_TOKEN!

export function GlobalMap() {
  const mapContainer = useRef<HTMLDivElement>(null)
  const map = useRef<mapboxgl.Map | null>(null)
  const isMapLoaded = useRef(false)
  
  const { 
    activeDisease, 
    activeYear, 
    activeScenario,
    selectedCountrySlug,
    isProjected,
    showNewlyAtRiskLayer,
    selectCountry
  } = useMapStore()
  
  const { data: mapData, isLoading } = useMapData(
    activeDisease, 
    activeYear, 
    isProjected ? activeScenario : undefined
  )
  
  // Initialize map
  useEffect(() => {
    if (!mapContainer.current || map.current) return
    
    map.current = new mapboxgl.Map({
      container: mapContainer.current,
      style: MAPBOX_STYLE_URL,
      center: [20, 20],
      zoom: 1.8,
      minZoom: 1,
      maxZoom: 8,
      projection: 'naturalEarth',  // Proper world map projection
      antialias: true,
    })
    
    map.current.on('load', () => {
      isMapLoaded.current = true
      initializeLayers()
    })
    
    return () => {
      map.current?.remove()
      map.current = null
    }
  }, [])
  
  const initializeLayers = useCallback(() => {
    if (!map.current) return
    
    // Country fill layer (data-driven coloring)
    map.current.addSource('countries', {
      type: 'geojson',
      data: { type: 'FeatureCollection', features: [] },
      generateId: true,
    })
    
    map.current.addLayer({
      id: 'country-fill',
      type: 'fill',
      source: 'countries',
      paint: {
        'fill-color': buildRiskColorExpression(),
        'fill-opacity': [
          'case',
          ['boolean', ['feature-state', 'hover'], false],
          0.85,
          0.70
        ],
      }
    })
    
    // Country border layer
    map.current.addLayer({
      id: 'country-border',
      type: 'line',
      source: 'countries',
      paint: {
        'line-color': '#2A3358',
        'line-width': 0.5,
      }
    })
    
    // Newly-at-risk pulsing border layer
    map.current.addLayer({
      id: 'country-new-risk-border',
      type: 'line',
      source: 'countries',
      filter: ['==', ['get', 'is_newly_at_risk'], true],
      paint: {
        'line-color': '#F5A623',
        'line-width': 2,
        'line-opacity': [
          'interpolate',
          ['linear'],
          ['get', 'suitability_score'],
          0.4, 0.5,
          1.0, 1.0,
        ],
      }
    })
    
    // Hover interaction
    let hoveredCountryId: string | null = null
    
    map.current.on('mousemove', 'country-fill', (e) => {
      if (e.features && e.features.length > 0) {
        if (hoveredCountryId !== null) {
          map.current!.setFeatureState(
            { source: 'countries', id: hoveredCountryId },
            { hover: false }
          )
        }
        hoveredCountryId = e.features[0].id as string
        map.current!.setFeatureState(
          { source: 'countries', id: hoveredCountryId },
          { hover: true }
        )
        map.current!.getCanvas().style.cursor = 'pointer'
      }
    })
    
    map.current.on('mouseleave', 'country-fill', () => {
      if (hoveredCountryId !== null) {
        map.current!.setFeatureState(
          { source: 'countries', id: hoveredCountryId },
          { hover: false }
        )
        hoveredCountryId = null
      }
      map.current!.getCanvas().style.cursor = ''
    })
    
    // Click interaction
    map.current.on('click', 'country-fill', (e) => {
      if (e.features && e.features.length > 0) {
        const slug = e.features[0].properties?.slug
        if (slug) selectCountry(slug)
      }
    })
    
  }, [selectCountry])
  
  // Update map data when disease/year/scenario changes
  useEffect(() => {
    if (!map.current || !isMapLoaded.current || !mapData) return
    
    const source = map.current.getSource('countries') as mapboxgl.GeoJSONSource
    if (source) {
      source.setData(mapData)
    }
  }, [mapData])
  
  // Animate newly-at-risk border opacity
  useEffect(() => {
    if (!map.current || !isMapLoaded.current) return
    
    let startTime: number
    let animFrame: number
    
    const animate = (timestamp: number) => {
      if (!startTime) startTime = timestamp
      const elapsed = (timestamp - startTime) / 2000  // 2 second cycle
      const opacity = 0.6 + 0.4 * Math.sin(elapsed * Math.PI * 2)
      
      map.current?.setPaintProperty(
        'country-new-risk-border',
        'line-opacity',
        showNewlyAtRiskLayer ? opacity : 0
      )
      
      animFrame = requestAnimationFrame(animate)
    }
    
    animFrame = requestAnimationFrame(animate)
    return () => cancelAnimationFrame(animFrame)
  }, [showNewlyAtRiskLayer])
  
  return (
    <div className="relative w-full h-[calc(100vh-64px)] overflow-hidden">
      {/* Map container */}
      <div ref={mapContainer} className="absolute inset-0" />
      
      {/* Loading overlay */}
      {isLoading && (
        <div className="absolute inset-0 bg-background-primary/30 
                        flex items-center justify-center">
          <div className="text-text-secondary text-sm font-mono">
            Loading {activeYear} data...
          </div>
        </div>
      )}
      
      {/* Projected data indicator */}
      {isProjected && (
        <div className="absolute top-20 left-1/2 -translate-x-1/2 
                        bg-accent-400/10 border border-accent-400/30 
                        text-accent-400 text-xs font-medium px-3 py-1.5 
                        rounded-full backdrop-blur-sm">
          PROJECTED DATA — {activeScenario} Scenario
        </div>
      )}
      
      {/* Left disease selector panel */}
      <DiseaseSelector />
      
      {/* Right country panel */}
      <CountryPanel />
      
      {/* Bottom time slider */}
      <TimeSlider />
      
      {/* Scenario toggle (only when projected) */}
      {isProjected && <ScenarioToggle />}
      
      {/* Map legend */}
      <MapLegend />
      
      {/* Zoom/pan controls */}
      <MapControls />
    </div>
  )
}

// src/components/map/TimeSlider.tsx
// Custom time slider with play/pause animation

'use client'

import { useEffect, useRef, useCallback } from 'react'
import { useMapStore } from '@/hooks/useMapStore'
import { Play, Pause, SkipBack } from 'lucide-react'

const MIN_YEAR = 1981
const MAX_YEAR = 2050
const PROJECTED_START = 2024

const PLAY_SPEEDS = {
  slow: 800,
  normal: 500,
  fast: 250,
}

export function TimeSlider() {
  const { 
    activeYear, 
    isPlaying, 
    playSpeed,
    setYear, 
    startPlaying, 
    stopPlaying,
    advanceYear,
    setPlaySpeed
  } = useMapStore()
  
  const intervalRef = useRef<NodeJS.Timeout>()
  const sliderRef = useRef<HTMLInputElement>(null)
  
  // Play animation
  useEffect(() => {
    if (isPlaying) {
      intervalRef.current = setInterval(
        advanceYear, 
        PLAY_SPEEDS[playSpeed]
      )
    } else {
      clearInterval(intervalRef.current)
    }
    
    return () => clearInterval(intervalRef.current)
  }, [isPlaying, playSpeed, advanceYear])
  
  const handleSliderChange = useCallback((e: React.ChangeEvent<HTMLInputElement>) => {
    stopPlaying()
    setYear(parseInt(e.target.value))
  }, [stopPlaying, setYear])
  
  const handleReset = useCallback(() => {
    stopPlaying()
    setYear(MIN_YEAR)
  }, [stopPlaying, setYear])
  
  const togglePlay = useCallback(() => {
    if (activeYear >= MAX_YEAR) setYear(MIN_YEAR)
    isPlaying ? stopPlaying() : startPlaying()
  }, [isPlaying, activeYear, startPlaying, stopPlaying, setYear])
  
  // Calculate slider progress percentage
  const pct = ((activeYear - MIN_YEAR) / (MAX_YEAR - MIN_YEAR)) * 100
  const projectedPct = ((PROJECTED_START - MIN_YEAR) / (MAX_YEAR - MIN_YEAR)) * 100
  
  return (
    <div className="absolute bottom-0 left-0 right-0 
                    bg-background-secondary/90 backdrop-blur-md
                    border-t border-border-subtle
                    px-6 py-4">
      
      {/* Year display */}
      <div className="flex items-center justify-between mb-3">
        <div className="flex items-center gap-3">
          {/* Play/Pause */}
          <button
            onClick={togglePlay}
            className="w-8 h-8 rounded-full bg-accent-400 hover:bg-accent-300
                       text-background-primary flex items-center justify-center
                       transition-colors duration-150"
            aria-label={isPlaying ? 'Pause' : 'Play'}
          >
            {isPlaying ? (
              <Pause size={14} strokeWidth={2.5} />
            ) : (
              <Play size={14} strokeWidth={2.5} fill="currentColor" />
            )}
          </button>
          
          {/* Reset */}
          <button
            onClick={handleReset}
            className="w-8 h-8 rounded-full border border-border-default 
                       text-text-secondary hover:text-text-primary hover:border-border-strong
                       flex items-center justify-center transition-colors duration-150"
            aria-label="Reset to 1981"
          >
            <SkipBack size={14} />
          </button>
          
          {/* Speed selector */}
          <div className="flex items-center gap-1 ml-2">
            {(['slow', 'normal', 'fast'] as const).map(speed => (
              <button
                key={speed}
                onClick={() => setPlaySpeed(speed)}
                className={`text-xs px-2 py-1 rounded transition-colors duration-150 ${
                  playSpeed === speed
                    ? 'bg-border-default text-text-primary'
                    : 'text-text-tertiary hover:text-text-secondary'
                }`}
              >
                {speed === 'slow' ? '0.5×' : speed === 'normal' ? '1×' : '2×'}
              </button>
            ))}
          </div>
        </div>
        
        {/* Current year display */}
        <div className="font-mono text-2xl font-bold text-text-primary">
          {activeYear}
          {activeYear > 2023 && (
            <span className="text-accent-400 text-sm font-normal ml-2">
              PROJECTED
            </span>
          )}
        </div>
        
        {/* Year range labels */}
        <div className="flex gap-4 text-text-tertiary text-xs font-mono">
          <span>{MIN_YEAR}</span>
          <span>TODAY: {PROJECTED_START}</span>
          <span>{MAX_YEAR}</span>
        </div>
      </div>
      
      {/* Slider track */}
      <div className="relative h-6 flex items-center">
        {/* Background track */}
        <div className="absolute w-full h-1 bg-border-default rounded-full" />
        
        {/* Historical fill */}
        <div 
          className="absolute h-1 bg-accent-400 rounded-full transition-none"
          style={{ width: `${Math.min(pct, projectedPct)}%` }}
        />
        
        {/* Projected fill (dashed effect via repeating gradient) */}
        {pct > projectedPct && (
          <div 
            className="absolute h-1 rounded-r-full"
            style={{ 
              left: `${projectedPct}%`,
              width: `${pct - projectedPct}%`,
              background: `repeating-linear-gradient(
                90deg, 
                #F5A623 0px, #F5A623 4px, 
                transparent 4px, transparent 8px
              )`
            }}
          />
        )}
        
        {/* TODAY marker */}
        <div 
          className="absolute flex flex-col items-center"
          style={{ left: `${projectedPct}%`, transform: 'translateX(-50%)' }}
        >
          <div className="w-px h-3 bg-text-tertiary" />
          <span className="text-text-tertiary text-xs mt-1">NOW</span>
        </div>
        
        {/* Native range input (invisible, handles drag) */}
        <input
          ref={sliderRef}
          type="range"
          min={MIN_YEAR}
          max={MAX_YEAR}
          value={activeYear}
          onChange={handleSliderChange}
          className="absolute w-full opacity-0 cursor-pointer h-6 z-10"
          aria-label="Select year"
        />
        
        {/* Custom thumb */}
        <div 
          className="absolute w-8 h-8 rounded-full bg-accent-400 
                     border-2 border-background-primary
                     flex items-center justify-center
                     shadow-glow-accent pointer-events-none
                     -translate-x-1/2 transition-none"
          style={{ left: `${pct}%` }}
        >
          <span className="text-background-primary text-xs font-mono font-bold leading-none">
            {String(activeYear).slice(2)}
          </span>
        </div>
      </div>
    </div>
  )
}

# etl/processors/geojson_builder.py

import geopandas as gpd
import json
from shapely.geometry import mapping
import topojson

def build_countries_geojson():
    """
    Build optimized GeoJSON for Mapbox.
    
    Optimizations:
    1. Simplify polygon geometries (reduce vertices)
    2. Limit decimal precision to 3 places
    3. Minimize properties (only what Mapbox needs)
    4. Split into world + detail levels
    """
    # Load Natural Earth 110m (for world view, zoom 1-4)
    world_low = gpd.read_file('natural_earth/ne_110m_admin_0_countries.shp')
    
    # Load Natural Earth 50m (for regional view, zoom 4-7)  
    world_med = gpd.read_file('natural_earth/ne_50m_admin_0_countries.shp')
    
    # Simplify geometries (tolerance in degrees)
    world_low.geometry = world_low.geometry.simplify(
        tolerance=0.5, 
        preserve_topology=True
    )
    world_med.geometry = world_med.geometry.simplify(
        tolerance=0.1,
        preserve_topology=True
    )
    
    # Keep only necessary columns
    keep_cols = ['ISO_A3', 'ISO_A2', 'NAME', 'geometry']
    world_low = world_low[keep_cols]
    world_med = world_med[keep_cols]
    
    # Save as GeoJSON with precision=3
    world_low.to_file(
        'processed/countries_world.geojson',
        driver='GeoJSON',
        precision=3
    )
    world_med.to_file(
        'processed/countries_regional.geojson',
        driver='GeoJSON',
        precision=4
    )
    
    # Validate file sizes
    for fname in ['countries_world.geojson', 'countries_regional.geojson']:
        size_mb = Path(f'processed/{fname}').stat().st_size / 1e6
        print(f"{fname}: {size_mb:.1f} MB")
        assert size_mb < 3.0, f"{fname} is too large: {size_mb}MB"
    
    print("GeoJSON built successfully")

      monthly_precip: [80, 80, 80, 80, 0, 0, 0, 0, 0, 0, 0, 0],
      altitude_median: 100
    }, 'dengue')
    
    expect(result.suitability_score).toBe(0)
    expect(result.temp_score).toBe(0)
  })
  
  it('returns high score for optimal tropical conditions', () => {
    const result = calculateSuitability({
      temp_mean_annual: 28,   // Perfect dengue temperature
      temp_min_coldest: 22,   // Warm year-round
      humidity_mean: 80,
      monthly_precip: [100, 100, 100, 100, 100, 100, 100, 100, 100, 100, 100, 100],
      altitude_median: 50
    }, 'dengue')
    
    expect(result.suitability_score).toBeGreaterThan(0.8)
    expect(result.risk_level).toBe('critical')
  })
  
  it('returns 0 winter_score when coldest month is below threshold', () => {

    }, 'dengue')
    
    expect(result.winter_score).toBe(0)
  })

14.3 API Health Check
TypeScript

// src/app/api/health/route.ts

import { NextResponse } from 'next/server'
import { createServerSupabaseClient } from '@/lib/supabase/server'

export async function GET() {
  const start = Date.now()
  
  try {
    const supabase = createServerSupabaseClient()
    
    // Simple query to verify DB connection
    const { error } = await supabase
      .from('data_freshness')
      .select('source_name, last_updated, status')
      .limit(5)
    
    if (error) throw error
    
    return NextResponse.json({
      status: 'healthy',
      database: 'connected',
      response_ms: Date.now() - start,
      timestamp: new Date().toISOString()
    })
    
  } catch (error) {
    return NextResponse.json({
      status: 'degraded',
      database: 'error',
      error: 'Database connection failed',
      timestamp: new Date().toISOString()
    }, { status: 503 })
  }
}

15.2 Scaling Architecture (Post-MVP)
When traffic requires scaling beyond free tiers:

text

Level 1 (1K-10K users/month):
  → Stay on free tiers, optimize queries
  → Add Cloudflare CDN caching for API responses
  → Cost: $0

Level 2 (10K-100K users/month):
  → Upgrade Supabase to Pro ($25/month)
  → Upgrade Vercel to Pro ($20/month)
  → Cost: ~$45/month

Level 3 (100K+ users/month):
  → Supabase Pro with connection pooler
  → Vercel Pro with higher function limits
  → Add Redis caching layer (Upstash Pro)
  → Cost: ~$100-200/month

Level 4 (Enterprise):
  → Dedicated database servers
  → Multi-region deployment
  → Enterprise partnerships with WHO, governments
  → Cost: Self-sustaining from revenue
